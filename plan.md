# Project Plan — Chatting App

## Entities

1. **User**
2. **Post**
3. **Story**
4. **Conversation** (1-to-1 chat)
5. **Message**
6. **GroupChat**
7. **GroupMember**
8. **GroupMessage**

---

## Functional Requirements (User Stories)

### User
- User can Register Account
- User can View Profile
- User can Update Profile
- User can Delete Account
- User can Upload ProfileImage

### Post
- User can Create Post
- User can View Post
- User can View Feed
- User can Delete Post

### Story
- User can Create Story
- User can View ActiveStories
- User can Delete Story

### Conversation (1-to-1)
- User can Create Conversation
- User can View Conversation
- User can View UserConversations

### Message
- User can Send Message
- User can View ConversationMessages
- User can Update Message
- User can Delete Message

### GroupChat
- User can Create GroupChat
- User can View GroupChat
- User can View UserGroupChats

### GroupMember
- User can Join GroupChat
- User can Leave GroupChat
- User can View GroupMembers

### GroupMessage
- User can Send GroupMessage
- User can View GroupMessages
- User can Update GroupMessage
- User can Delete GroupMessage
